Project Title :> Heart Disease Prediction using Machine Learning

Problem:
Heart disease is one of the leading causes of death worldwide.
This project predicts the risk of heart disease using machine learning models.

Dataset
Heart Disease Dataset

Exploratory Data Analysis:
EDA was performed to understand:
age distribution
cholesterol levels
heart rate patterns
correlation between variables

Models Tested:
Logistic Regression
KNN
Naive Bayes
SVM
Decision Tree

Model Comparison:
Model:                               	Accuracy	            F1 Score
Logistic Regression                  	0.8947	               0.9167
KNN                                  	0.8421	               0.8696
Naive Bayes                         	0.8421                 0.8696
SVM	                                  0.8947	               0.9167
Decision Tree	                        0.7895	               0.8333


Final model used: Support Vector Machine

Web Application:
A web interface was built using Streamlit where users can input health parameters to predict heart disease risk.
